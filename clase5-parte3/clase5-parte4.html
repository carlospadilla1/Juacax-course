Rem y Em

Rem: Esta unidad se basa en el tamaño de fuente del elemento raíz (HTML) y, por defecto, es igual a 16px if no se establece en el navegador del usuario. Cada elemento secundario que utilice rem utilizará el tamaño de fuente raíz HTML como punto de cálculo, independientemente de si un elemento padre tiene tamaños diferentes especificados.

Em: En contraste, esta unidad se basa en el tamaño de fuente del elemento principal. Si un elemento principal tiene un tamaño diferente al del elemento raíz, el cálculo de em se basará en el elemento principal y no en el elemento raíz. Esto significa que los elementos anidados que utilizan em para el tamaño a veces pueden terminar teniendo un tamaño que no se anticipó. Sin embargo, proporciona un control más detallado si es necesario especificar el tamaño de un área particular de una página.


Nota: El HTML viene por defecto con 16px para las medidas de las fuentes.
